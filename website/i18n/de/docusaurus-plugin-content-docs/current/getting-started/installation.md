# Installation

IaC Code erfordert Python 3.10 oder neuer.

Installieren Sie das Paket aus dem konfigurierten Python-Paketindex:

```bash
pip install iac-code
```

Ueberpruefen Sie den Befehl:

```bash
iac-code --help
```
