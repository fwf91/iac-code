# 設定

IaC Code は CLI 引数、環境変数、ランタイム設定ディレクトリのファイルから設定を読み取ります。

設定の優先順位：

```text
CLI 引数 > 環境変数 > 設定ファイル
```

ランタイムディレクトリ：

```text
~/.iac-code/
```

主なファイル：

| ファイル | 説明 |
|---|---|
| `.credentials.yml` | LLM 認証情報 |
| `.cloud-credentials.yml` | クラウドプロバイダー認証情報 |
| `settings.yml` | 選択されたプロバイダー、モデル、関連設定 |
| 履歴ファイル | 対話ワークフローの入力履歴 |

このディレクトリのファイルはシークレットやローカル設定を含む可能性があるため、コミットや共有を避けてください。
