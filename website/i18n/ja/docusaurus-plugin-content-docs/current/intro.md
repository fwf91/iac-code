# 概要

IaC Code は、Alibaba Cloud 向けの AI 駆動 Infrastructure as Code アシスタントです。ターミナルワークフローを通じて、ROS および Terraform テンプレートの生成、デプロイ、管理をサポートします。

主な機能：

- **言葉にすれば、即生成** — 自然言語で必要なものを記述するだけで、検証済みですぐにデプロイ可能な ROS または Terraform テンプレートを生成します。
- **ワンコマンドで本番へ** — テンプレートから稼働中のインフラとアプリケーションまでを一気通貫で実現。リージョンをまたいでスタックの作成・更新・削除・監視を行います。
- **クラウドの知見を内蔵** — ドキュメント検索、リソース在庫確認、デプロイ前のコスト見積もり。すべての判断が実際のクラウドデータに裏付けられています。

ドキュメントはユーザータスクに沿って構成されています。まずインストールとクイックスタートから始め、次にプロバイダーと認証情報を設定し、コマンドの詳細が必要な場合は CLI リファレンスをご覧ください。
